# 欧拉图 / 哈密顿图判断器

一个可视化的图论工具，用于判断图是否为欧拉图或哈密顿图，并计算使其成为欧拉图/哈密顿图所需的最少边数。

## 功能特性

- ✅ **欧拉图判断**：判断是否存在欧拉回路或欧拉通路
- ✅ **哈密顿图判断**：判断是否存在哈密顿回路、哈密顿通路或半哈密顿图
- ✅ **有向/无向图**：支持有向图和无向图的切换
- ✅ **可视化界面**：直观的图形化操作界面，支持网格布局和自由布局
- ✅ **边和节点操作**：
  - 支持平行边（最多3条）
  - 支持自环（最多3个）
  - 支持添加/删除节点
- ✅ **动画演示**：可播放欧拉通路/回路和哈密顿通路/回路的遍历动画
- ✅ **预设图形**：包含经典彼得森图和广义彼得森图
- ✅ **边的命名**：自动为边命名（e1, e2, e3...），可选择性显示

## 系统要求

- Python 3.6 或更高版本
- tkinter（通常随 Python 一起安装）

## 安装与运行

1. 克隆仓库：
```bash
git clone https://github.com/XueLanBaiLan/euler-hamilton-graph-checker.git
cd euler-hamilton-graph-checker
```

2. 运行程序：
```bash
python app.py
```

## 使用说明

### 基本操作

1. **创建网格**：在界面中设置网格的行数和列数，点击"重建网格"
2. **添加边**：点击两个节点可以添加边（最多3条平行边）
3. **自环**：点击同一个节点两次可以添加自环
4. **删除边**：再次点击已连接的节点可以删除边
5. **切换模式**：使用"有向模式"复选框切换有向图/无向图

### 高级功能

- **添加节点**：勾选"添加节点模式"，在空白处点击可添加新节点
- **删除节点**：勾选"删除节点模式"，点击节点可删除节点及其关联边
- **显示边名**：勾选"显示边名"可以查看每条边的名称
- **预设图形**：使用"经典彼得森图"或"广义彼得森图"按钮快速创建预设图形

### 判断功能

点击"检查"按钮后，程序会：
- 判断图是否为欧拉图/半欧拉图/哈密顿图/半哈密顿图
- 计算使其成为欧拉图/哈密顿图所需的最少边数
- 提供可视化结果窗口，可选择不同的欧拉/哈密顿路径
- 支持播放路径遍历动画，可调节动画速度

## 算法说明

- **欧拉图判断**：基于顶点度数和图的连通性（有向图考虑入度/出度）
- **哈密顿图判断**：使用动态规划算法（适用于顶点数 ≤ 20 的图）
- **路径查找**：使用 DFS 和 DP 相结合的方法
